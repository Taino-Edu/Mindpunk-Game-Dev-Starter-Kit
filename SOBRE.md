#  Mindpunk Game Dev Starter Kit

Bem-vindo ao **Mindpunk Game Dev Starter Kit**! Este é um projeto introdutório e open-source criado pela [Mindpunk Lab] para auxiliar novos entusiastas a entrarem no mundo do desenvolvimento de jogos.

A ideia aqui é quebrar o mito de que você precisa de engines complexas (como Unity ou Unreal) para começar. Vamos te mostrar que lógica é lógica, e você pode criar jogos incríveis usando ferramentas do dia a dia como JavaScript, Terminal, Express e Angular.

##  A Filosofia Mindpunk
Na Mindpunk Lab, acreditamos em jogos táticos, mecânicas sólidas e, acima de tudo, em **colocar a ideia no papel antes de codar**. Todo grande jogo começa com um bom planejamento e um escopo bem definido.

##  O que você vai encontrar aqui

*  **`/docs`**: A teoria que salva vidas. Aqui você encontra nosso template de **GDD (Game Design Document) Lite**.
* 🎮 **`/jogos-exemplos`**: O código fonte! Projetos dissecados para você entender a lógica:
  * Jogo em Terminal (JS Puro)
  * Jogo com Backend (Express)
  * Jogo com Interface (Angular)
*  **`/desafios`**: Onde a mágica acontece. Exercícios práticos para você testar o que aprendeu.

##  Como usar este repositório

1. Faça o clone deste repositório na sua máquina.
2. Leia o nosso guia de GDD na pasta `/docs`.
3. Explore os códigos na pasta `/jogos-exemplos` (cada pasta tem instruções de como rodar).
4. Assista aos vídeos de apoio linkados em cada seção (em breve!).

##  Mão na Massa (Desafios)
Aprender a programar jogos exige prática. Na pasta `/desafios` você encontrará propostas para modificar os códigos de exemplo. Conseguiu resolver um? Faça um *Fork* deste repositório, adicione sua solução e abra um *Pull Request*!
