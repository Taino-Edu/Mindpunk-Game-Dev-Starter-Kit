# Configuração da Logo Mindpunk

O README foi atualizado para incluir a logo de forma profissional e discreta.

## Como Colocar a Logo

1. Copie o arquivo da logo Mindpunk
2. Coloque em: `assets/logos/mindpunk-logo.png`
3. Faça commit e push

A estrutura deve ficar assim:
```
Mindpunk-Game-Dev-Starter-Kit/
├── assets/
│   └── logos/
│       └── mindpunk-logo.png  (sua logo aqui)
├── README.md
└── ...
```

## Como o README Referencia a Logo

O README agora tem no topo:
```markdown
<div align="center">
<img src="assets/logos/mindpunk-logo.png" alt="MINDPUNK" width="200" style="margin-bottom: 20px;">

# MINDPUNK - Game Development Education Kit
</div>
```

A logo aparece:
- Centralizada
- Com 200px de largura (proporção elegante)
- Acima do título principal
- Discreta e profissional (não agressiva)

## Requisitos da Logo

- Formato: PNG (transparente de preferência)
- Tamanho recomendado: 400x400px ou maior
- Aspecto: Logo Mindpunk (a que você enviou)

Depois que colocar a logo, o README ficará completo e profissional para recrutadores.
