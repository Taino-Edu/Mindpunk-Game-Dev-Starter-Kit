# ⚔️ Micros vs Macros - Estratégia

---

## O Que São Micros e Macros?

Essas palavras vêm de jogos de estratégia e significam:

### **MICRO** (Ações Pequenas)
```
Foco: AGORA, ESTA UNIDADE

Exemplos de ações MICRO:
├─ Mover UM soldado 2 casas pra esquerda
├─ Atacar UM inimigo
├─ Usar habilidade especial de UM soldado
└─ Coletar recurso com UM trabalhador

Pergunta MICRO:
"Eu movo esse soldado pra cá ou pra lá?"
```

### **MACRO** (Estratégia Geral)
```
Foco: PLANO GERAL, TODO O JOGO

Exemplos de ações MACRO:
├─ Onde posicionar TODAS as minhas tropas?
├─ Qual é minha rota de ataque?
├─ Devo atacar agora ou esperar pra poupar recursos?
└─ Como vou defender minha base?

Pergunta MACRO:
"Qual é minha estratégia para vencer?"
```

## 🎮 Exemplo: Uma Batalha Real

```
SITUAÇÃO:
Você tem:
  ├─ 2 infantarias
  ├─ 1 tanque
  └─ 1 arqueiro

Inimigo tem:
  ├─ 3 infantarias
  └─ 1 tanque

Mapa: 10x10
```

---

### NIVEL MICRO: O Que Fazer Agora?

```
Turno 1:
  └─ "Movo o ARQUEIRO 2 casas pra cima?"
  └─ "Ou movo o TANQUE pra frente?"

Turno 2:
  └─ "Meu tanque ataca o tanque inimigo?"
  └─ "Ou meu infantaria ataca a infantaria?"

Turno 3:
  └─ "Meu arqueiro atira de longe?"
  └─ "Ou recua pra segurança?"
```

**Resposta MICRO:**
- Cada ação é tomada individualmente
- Foco: Qual movimento AGORA é melhor?

---

### NÍVEL MACRO: Qual é o Plano?

```
ANTES DO JOGO:

Estratégia 1: "ATAQUE FRONTAL"
├─ Coloco tudo na frente
├─ Ataco direto no inimigo
└─ Espero derrotar antes de sofrer muito dano

Estratégia 2: "FLANKEIO"
├─ Divido minha força
├─ Uma parte segura, outra ataca pela lateral
└─ Inimigo fica confuso e derroto

Estratégia 3: "DEFESA"
├─ Uso meu tanque como escudo
├─ Arqueiro ataca de trás
└─ Deixo inimigo gastar recursos me atacando
```

**Resposta MACRO:**
- Você escolhe UMA estratégia geral
- Todas as ações MICRO devem servir esse plano

---

## 🎯 Como Micros E Macros Trabalham Juntos

```
MACRO: "Vou fazer um FLANKEIO"
  ├─ Divido minhas tropas em 2 grupos
  └─ Um vai pela esquerda, outro pela direita

MICRO: Implementando a estratégia
  ├─ Turno 1: Movo grupo 1 três casas pra esquerda
  ├─ Turno 2: Movo grupo 2 três casas pra direita
  ├─ Turno 3: Grupo 1 ataca pela lateral
  └─ Turno 4: Grupo 2 ataca pela frente

RESULTADO:
  ✅ Estratégia funcionou
  ✅ Inimigo foi derrotado
```

---

## 💡 Erro Comum: Micro Sem Macro

```
❌ "Vou ficar mudando de ideia a cada turno"

Turno 1: "Ataco pela frente!"
Turno 2: "Não, vou flanquear!"
Turno 3: "Na verdade, vou pra defesa!"

RESULTADO:
  ❌ Suas tropas fica desorganizadas
  ❌ Inimigo vence facilmente
```

## 🏗️ Como Construir Sua Estratégia (MACRO)

### Passo 1: Analise a Situação

```
Pergunta: "Qual é meu objetivo?"
├─ Matar o inimigo?
├─ Proteger um ponto?
├─ Conseguir recursos?
└─ Outra coisa?

Pergunta: "Qual é minha força?"
├─ Tenho mais tropas?
├─ Minhas tropas são melhores?
├─ Tenho posição melhor?
└─ Qual é minha vantagem?

Pergunta: "Qual é a fraqueza do inimigo?"
├─ Ele tem poucos arqueiros?
├─ Sua defesa é fraca?
├─ Ele é lento?
└─ Como explorar isso?
```

### Passo 2: Escolha Uma Estratégia

```
Se TENHO VANTAGEM numérica:
  └─ Escolher: ATAQUE FRONTAL
     └─ Usar números pra vencer

Se TENHO VANTAGEM de posição:
  └─ Escolher: FLANKEIO
     └─ Atacar pela lateral, inimigo fica confuso

Se TENHO VANTAGEM de tipo de unidade:
  └─ Escolher: FOCO EM FRAQUEZA
     └─ Meus arqueiros exploram infantaria fraca dele

Se ESTOU EM DESVANTAGEM:
  └─ Escolher: DEFESA
     └─ Proteger, desgastar inimigo, tentar vencer nos detalhes
```

### Passo 3: Execute os Micros Conforme o Plano

```
Seu plano: FLANKEIO
  └─ Cada turno (MICRO):
     ├─ Grupo 1 → se move pra esquerda
     ├─ Grupo 2 → se move pra frente
     └─ Quando pronto → ATAQUE COORDENADO
```

## 📊 Exemplo Real em Game-UW

```
TURNO 1-3: (MACRO - Preparação)
Você posiciona:
  ├─ Tanque no centro (proteção)
  ├─ Infantaria à esquerda (flanco)
  └─ Arqueiro atrás (apoio)

Sua estratégia: DEFESA + CONTRA-ATAQUE

TURNO 4-6: (MICRO - Execução)
Inimigo ataca seu tanque:
  ├─ Turno 4: Tanque recebe dano (-20 HP)
  ├─ Turno 5: Tanque recebe mais dano (-15 HP)
  ├─ Turno 6: Arqueiro contra-ataca de trás (-30 HP inimigo)

TURNO 7: (MACRO - Decisão Crítica)
Inimigo está fraco:
  └─ Você mudar pra CONTRA-ATAQUE AGRESSIVO
     └─ Infantaria e tanque atacam junto

RESULTADO:
  ✅ Inimigo derrotado
```

## ✅ Resumo: O Que Você Aprendeu

- [ ] **MICRO** = Ações pequenas (mover uma unidade, atacar uma)
- [ ] **MACRO** = Estratégia geral (qual é meu plano?)
- [ ] Bons micros sem macro = derrota
- [ ] Boa macro sem bons micros = derrota
- [ ] **Você precisa dos dois!**
